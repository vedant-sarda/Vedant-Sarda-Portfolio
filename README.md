# Vedant Sarda — Portfolio (Next.js)

  Dark, animated portfolio built with **Next.js 14 (App Router) + JavaScript + TailwindCSS v4 + Framer Motion**.

  ## Run locally

  Requires **Node.js 18.18+** (Node 20+ recommended).

  ```bash
  npm install
  npm run dev
  ```

  Open http://localhost:3000

  ## Build & start

  ```bash
  npm run build
  npm start
  ```

  ## Edit content

  - Personal info / socials → `src/data/constants.js`
  - Projects → `src/data/projects.js`
  - Skills → `src/data/skills.js`
  - Certifications → `src/data/certifications.js`
  - Education → `src/data/education.js`
  - Resume PDF → `public/Vedant_Sarda_2026.pdf`
  - Resume preview → `public/resume-preview.png`
